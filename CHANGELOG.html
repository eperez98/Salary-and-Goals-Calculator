🐛 Critical Bug Fixed in v2.0 (was not in v1.x)
  • FIXSilent crash on launch — dual ctk.CTk() root: CalcApp inherited ctk.CTk, so calling CalcApp() from the entry point created a second Tk interpreter while the first was already in mainloop(). Windows silently killed the process with no error output. Fixed by changing CalcApp(ctk.CTk) → CalcApp(ctk.CTkFrame), adding a root parameter, and delegating all window-level calls (title, minsize, iconbitmap) to self.root.

Cancelled Versions
v1.1.1 — Planned Hotfix ⊘ Cancelled Never shipped
⊘ Why this version was cancelled v1.1.1 was started as a hotfix for two bugs found right after v1.1 shipped: a launch crash (AttributeError: _step_resultado) and the language popup not centering on screen. The fixes were developed and validated, but rather than shipping them as a patch on top of the aging plain-tkinter architecture, the decision was made to skip v1.1.1 entirely and fold all fixes directly into the v2.0 Modern Edition rewrite using customtkinter. The bugs were fully resolved in v2.0.
What was fixed instead in v2.0
  • FIXAttributeError: 'CalcApp' has no attribute '_step_resultado' — the def _step_resultado(self): declaration was accidentally dropped during v1.1 development. Method body existed but Python never registered the method. Fixed in v2.0 by full rewrite.
  • FIXLanguage popup not centered on screen — winfo_screenwidth() called before update_idletasks(), returning 1 on Windows Python 3.14. Fixed in v2.0 by caching screen dimensions once at startup before any UI operations, and migrating wait_window() to wait_variable(BooleanVar).
v0.3 Beta ⊘ Cancelled Never shipped
⊘ Why this version was cancelled v0.3 Beta was underway when repeated Python 3.14 crashes made it unshippable. The crashes originated from creating Toplevel windows before root.update() was called — a deep architectural flaw with no safe incremental patch. Rather than ship a broken release, all completed v0.3 work (Mexico taxes, canvas flags, Win11 themes, animated splash) was frozen and folded into a ground-up rebuild shipped directly as v1.0 RC.
